# Travel Planner using AVL Trees

## Overview

Travel Planner is a console-based application developed in C that helps users create and manage travel itineraries. The application stores travel activities such as flights, hotels, transportation, and tourist destinations while also maintaining navigation routes between locations.



---

## Features

- Create and manage complete travel itineraries
- Add, update and delete travel activities
- Maintain navigation routes between destinations
- Search travel routes efficiently
- Detect duplicate locations in a trip
- Display complete travel itinerary
- Sort travel records based on hotel charges
- Automatically maintain chronological order using AVL Trees

---

## Data Structures Used

- AVL Tree for Trip Activities
- AVL Tree for Navigation Records
- Modular C Programming
- Dynamic Memory Allocation

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Insert Activity | O(log n) |
| Delete Activity | O(log n) |
| Search Activity | O(log n) |
| Update Activity | O(log n) |
| Navigation Search | O(log n) |
